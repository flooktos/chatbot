# Conversation Rules

## Supported Behavior

- Follow-up Question
- Short Context Memory
- FAQ Matching
- Intent Switching

## Unsupported Behavior

- Open Conversation
- External Knowledge
- Real-time Information
- Free-form AI Answer

## Example

User: สมัครยังไง
Bot: ลูกค้าสามารถสมัครผ่านเว็บไซต์ของบริษัทได้ครับ

User: ใช้เอกสารอะไรบ้าง
Bot: ใช้บัตรประชาชนและเอกสารตามประเภทบริการครับ
